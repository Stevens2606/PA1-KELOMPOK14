# Quality Time Cafe & Resto

Website informasi dan pemesanan untuk Quality Time Cafe & Resto.

## Deskripsi
Quality Time Cafe & Resto adalah platform web yang menyediakan 
informasi lengkap tentang cafe dan resto, serta memudahkan 
pelanggan untuk melakukan pemesanan secara online.

## Fitur
- Informasi menu makanan dan minuman
- Pemesanan online
- Manajemen reservasi meja
- Informasi lokasi dan jam operasional

## Teknologi yang Digunakan
- Laravel (PHP Framework)
- MySQL
- Bootstrap
- HTML, CSS, JavaScript

## Cara Menjalankan
1. Clone repository ini
2. Jalankan `composer install`
3. Copy `.env.example` menjadi `.env`
4. Jalankan `php artisan migrate`
5. Jalankan `php artisan serve`
